# OpenClaw Digital Empire

Full-Stack AI Platform — Next.js 14 Command Center, FastAPI Backend, Multi-Platform Bots, Chess League.

## Architecture
- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS + Recharts
- **Backend**: FastAPI + Python 3.12 + asyncpg + Redis
- **Database**: Supabase Postgres
- **Bots**: Telegram x3, Discord, Slack
- **Infra**: Vercel (frontend) + Render (backend) + Supabase (DB)

## Quick Start
```bash
docker-compose up --build
```
